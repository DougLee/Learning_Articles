### 1. 并发编程的三个重要特性: 可见性, 原子性和有序性

#### 原子性

在一次或者多次操作中, 所有的操作要么全部执行, 要么全部不执行.

#### 可见性

可见性是指当一个线程对共享变量进行了修改, 那么另外的线程可以立即看到修改后的最新值.

#### 有序性

有序性是指程序代码在执行过程中的先后顺序未必就是开发者编写代码的顺序.(由于Java在编译器以及运行期的优化).即**指令重排序**.
